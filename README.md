# 🧠 Neural Mesh: Arte Generativa Dinâmica

![Tecnologia: HTML5/JS](https://img.shields.io/badge/Tecnologia-HTML5%20%2F%20JS-F7DF1E.svg)
![Estética: Cyberpunk](https://img.shields.io/badge/Estética-Cyberpunk-bc00ff.svg)

**Neural Mesh** é um experimento visual de arte generativa que simula o comportamento de uma rede neural abstrata. Desenvolvido com **HTML5 Canvas**, o projeto explora a conexão entre pontos aleatórios (neurônios) e a interatividade em tempo real com o usuário.

---

## 🚀 [VER DEMONSTRAÇÃO AO VIVO](https://brendowjanuzzi-ui.github.io/rede-neural-generativa/)

> **Interação:** Mova o mouse pela tela para atrair as conexões e criar distorções na malha neural.

---

## 🧪 Conceitos Aplicados

* **Plexus Effect:** Sistema de partículas onde linhas são desenhadas automaticamente entre pontos que atingem uma distância mínima de proximidade.
* **Física de Partículas:** Cada "nó" da rede possui vetores de velocidade individuais e lógica de colisão com as bordas da tela.
* **Interatividade Magnética:** O cursor do mouse atua como um campo de força, influenciando a posição e a opacidade das conexões próximas.
* **Design Minimalista:** Paleta de cores focada em alto contraste para uma estética tecnológica e limpa.

---

## 🛠️ Tecnologias Utilizadas

* **Canvas API:** Renderização de alto desempenho para centenas de objetos simultâneos.
* **JavaScript Puro (Vanilla):** Lógica matemática para cálculo de distância Euclidiana entre os pontos.
* **CSS Moderno:** Interface minimalista e responsiva.
